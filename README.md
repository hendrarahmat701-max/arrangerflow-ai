# ArrangerFlow AI: Arranger Construction Workstation

ArrangerFlow AI is a specialized workstation designed for keyboard arrangers and style creators. It focuses on **Style Construction**, **Adaptation**, and **Symbolic Workflow**, providing a lightweight yet powerful utility for the global arranger community.

## 🎹 Core Philisophy: Human-Guided System
Instead of full automation, ArrangerFlow AI puts the arranger in control:
- **Human Determines**: Groove, Chords, Sections, Rasa (Feel), and Genre.
- **System Assists**: Symbolic generation, mapping, adaptation, and workflow acceleration.

## 🚀 Key Modules

### 1. MP3 → STY (Workflow Assist)
A pipeline-driven approach to deconstructing references:
- **Audio Prep & Section Parsing**: Detect Intro, Main, and Fills.
- **Groove Alignment**: Extract and align rhythmic feel to project tempo.
- **AI Lead Detection**: Optional module for isolating melodies and providing draft notation.

### 2. Style Import System (Priority)
Breathe new life into existing content:
- **Import Existing Styles**: Parse MIDI lanes and Section markers.
- **Editable Reference**: Turn proprietary style markers into flexible workstation projects.
- **Remapping Engine**: Quickly shift instrument assignments to modern soundbanks.

### 3. SF2 Template Preparation
Specialized tool for managing SoundFonts:
- **Bank/MSB/LSB Analysis**: Deep inspection of SF2 files.
- **Role Assignment**: Define tracks as Drum, Bass, Lead, etc.
- **Genre Profiles**: Save mapping setups as `.sfprofile` for instant recall.

### 4. Section-Based Key System
Dynamic harmonic control per section (Intro in C, Main in Dm, Ending in G), ensuring maximum musical flexibility.

## 🛠 Technology Architecture
- **UI Layer**: React + Vite + TypeScript (Hardware-style specialist interface).
- **Core Strategy**: Offline-first for productivity, Online AI for complex extraction.
- **Low-Spec Optimized**: Designed for lightweight portability on standard arranger laptops.
